Polyclaw provides a rich set of capabilities beyond basic chat.

## Core Features

- [Messaging & Bot](/features/messaging/) -- Multi-channel messaging through Bot Framework
- [Realtime Voice](/features/voice/) -- Phone calls via Azure Communication Services
- [Scheduler & Proactive](/features/scheduler/) -- Cron tasks and autonomous messaging
- [Media Handling](/features/media/) -- File, image, audio, and video processing
- [Memory System](/features/memory/) -- Automatic conversation consolidation
- [Slash Commands](/features/commands/) -- Comprehensive command interface

## Hardening

- [Guardrails & HITL](/features/guardrails/) -- Defense-in-depth tool interception with configurable mitigation strategies and per-model autonomy levels
- [Agent Identity](/features/agent-identity/) -- Least-privilege managed identity with credential isolation between admin and agent runtime
- [Tool Activity](/features/tool-activity/) -- Enterprise audit dashboard with automated scoring, manual flagging, and CSV export
- [Runtime Isolation](/deployment/runtime-isolation/) -- Separated admin and agent runtime architecture with independent credentials
- [Sandbox Execution](/features/sandbox/) -- Isolated code execution in Azure Container Apps dynamic sessions
- [Monitoring](/features/monitoring/) -- OpenTelemetry integration with Azure Monitor for traces, metrics, and logs
